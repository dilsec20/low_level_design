# low_level_design

A step-by-step curriculum and practice repository for mastering Low-Level Design (LLD) and Object-Oriented Design in C++.

Includes implementations and interview questions for:
- SOLID Principles
- Creational Design Patterns
- Structural Design Patterns
- Behavioral Design Patterns
- Mini LLD Projects (Tic-Tac-Toe, Parking Lot, URL Shortener)
